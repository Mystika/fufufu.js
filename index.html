fufufu.js Lightweight Presentation Library
What is fufufu.js? fufufu(ふふふ).js is lightweight presentation library. It dosen`t need jQuery or anything else!
How to use it?
  1. put fufufu.js into your project and link it
  2. make some div.slide
  3. do fufufu();
Fragments Use data-fragindex attribute for fragments How easy! Isn't it?
