present.js Lightweight Presentation Library
What is present.js? present.js is lightweight web presentation library. It dosen`t need jQuery or anything else!
How to use it?
  1. put present.js into your project and link it
  2. make some div.slide
  3. do present();
Fragments Use data-fragindex attribute for fragments How easy! Isn't it?
